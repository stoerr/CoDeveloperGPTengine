Set GPT Secret

This interface allows you to set a 'gptsecret' cookie for debugging purposes.

Debugging hints:

When you want to try repeating the requests done by ChatGPT from the browser, you will get a 'service access token missing' header since the proper Authorization header is missing. If you want to debug the Co-Developer GPT engine, you could use this page to set a cookie containing the gptsecret, instead. For more debugging hints see Debugging.md in the project sources.

To set the cookie copy the value for the gptsecret you've chosen in $HOME/.cgptcodeveloperglobal/config.properties into the form below and submit it. Then you can use the browser to repeat requests.
